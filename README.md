# DADS5001-Mini_Project
This repository is a part of DADS5001 coursework at NIDA. T
he objective of this mini-project is to apply fundamental data analytics concepts, statistical techniques, and data visualization methods to real-world datasets.

The project covers the complete data analysis workflow, including:
- Data collection and preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization
- Interpretation of results

Through this project, analytical thinking and practical implementation skills are demonstrated using Python and relevant data science libraries.
